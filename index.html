Instalacja

Linux

Pobierz
  1. Kliknij „Pobierz” powyżej.
  2. Otwórz lokalizację pliku.
  3. Otwórz terminal.
  4. Wpisz: pkexec dpkg -i (na końcu jest spacja).
  5. Przeciągnij plik do terminalu.
  6. Kliknij Enter.
  7. Wpisz hasło administratora.

Windows

Pobierz
  1. Zainstaluj 7-Zip.
  2. Kliknij „Pobierz” powyżej.
  3. Otwórz plik.
  4. Uruchom plik „Sterntył.pl 1.0.0”.
